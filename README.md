# LMS Rust client

The lms rust client

[lms](https://gitlab.com/saxion.nl/42/lms42)


### Features
 - [X] Upload work
 - [X] Download work
 - [X] Download template
 - [X] Reorder file structure
 - [ ] Grade work
 - [X] Login
 - [X] Open work dir
 - [ ] Handle setup
 
### Extra features
 - [X] Change default editor
 - [X] Open work offline 
 
Avalible for `arm` and `x86` systems
